# MVP Roadmap

## Phase 1 — Current Foundation
- Premium responsive UI.
- URL, message, and fake-credit scanners.
- Risk scoring engine and evidence model.
- Mock threat intelligence and AI advisor service.
- Prisma data model for SaaS expansion.

## Phase 2 — Production Hardening
- Auth, RBAC, admin MFA, API keys, Redis rate limits.
- PostgreSQL persistence for scans, reports, and indicators.
- Email scanner and header parser.
- PDF reports and saved results.

## Phase 3 — Threat Intelligence
- WHOIS/domain age, SSL, DNS, redirect crawler.
- Google Safe Browsing, VirusTotal, PhishTank, URLScan.io.
- False-positive workflow and official brand verification.

## Phase 4 — Channels and Monetization
- Browser extension, Telegram/WhatsApp bots, mobile app.
- Premium/family plans, B2B API metering, enterprise white-label, education packages.
